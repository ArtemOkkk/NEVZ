# Информатика. Практическая работа: Базы данных

Проект представляет собой систему управления НЭВЗом.

## Структура проекта

- 'main.py' - основной скрипт вложения.
- 'NEVZ.db' - база данных SQL.
- 'SQL_Запросы/' - папка с SQL-скриптами для работы с базой.
- 'Photo/' - папка с изображениями.

- ## Требования

- Python 3.x
- SQlite
