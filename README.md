# Управление парком электровозов НЭВЗ — База данных

Практическая работа по информатике: «Проектирование, нормализация и работа с реляционными базами данных»

Проект представляет собой систему учёта выдачи локомотивов для Новочеркасского электровозостроительного завода (НЭВЗ), разработанную на SQLite. База данных содержит информацию о работниках, локомотивах, депо приписки и выдачах локомотивов.

## Технологии
- Язык: Python 3
- БД: SQLite
- Инструменты: SQLite Online, GitHub

## Структура базы данных

### Таблица workers (Работники)
| Поле | Тип | Описание |
|------|-----|----------|
| id | INTEGER | Уникальный идентификатор работника (Primary Key) |
| name | TEXT | ФИО сотрудника |
| post | TEXT | Должность |
| department | TEXT | Отдел / цех |
| work_experience | INTEGER | Стаж работы (лет) |
| salary | INTEGER | Заработная плата (₽) |

### Таблица locomotives (Локомотивы)
| Поле | Тип | Описание |
|------|-----|----------|
| id | INTEGER | Уникальный идентификатор локомотива (Primary Key) |
| series | TEXT | Серия электровоза |
| type | TEXT | Тип (пассажирский, грузовой, промышленный) |
| current_type | TEXT | Род тока (переменный, постоянный, двухсистемный) |
| number | INTEGER | Бортовой номер |
| power_kw | INTEGER | Мощность (кВт) |
| construction_speed | INTEGER | Конструкционная скорость (км/ч) |

### Таблица delivery_depot (Депо приписки)
| Поле | Тип | Описание |
|------|-----|----------|
| id | INTEGER | Уникальный идентификатор депо (Primary Key) |
| depot_name | TEXT | Название депо |
| railway | TEXT | Железная дорога |
| region | TEXT | Федеральный округ |
| city | TEXT | Город расположения |
